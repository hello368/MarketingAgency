## 2. 경쟁사 유형별 분석

### Type A: 전통 MedSpa 에이전시

**특징:**
- 광고 크리에이티브 + 캠페인 운영 중심
- 수동 리포팅 (주 1회 스프레드시트)
- 리드 전달 후 후속은 클리닉 책임
- 1 AM당 5-8 클라이언트

**가격:** $1,000-$3,000/월 플랫 리테이너

**약점:**
- CAPI 연동 없음 → 광고 최적화 한계
- 예약 자동화 없음 → 리드 85%+ 소실
- 리포트가 CPL 중심 (CPB 모름)
- 확장성 없음 (사람 비례)

**우리의 반박:**
> "그들은 리드를 드려요. 저희는 예약이 채워지는 시스템을 만들어요.
> 리드가 쌓여도 예약이 안 되면 광고비가 낭비되는 겁니다."

---

### Type B: 일반 GHL 에이전시 (자동화 전문)

**특징:**
- GHL 설치 + 기본 워크플로우 전문
- MedSpa만 하는 게 아닌 여러 업종 취급
- 자동화는 있지만 Meta CAPI 연동 약함
- Snapshot 판매 위주

**가격:** $500-$1,500/월 + GHL 라이선스

**약점:**
- MedSpa 시술별 광고 전략 없음
- 시즌 캘린더, 트렌드 인텔리전스 없음
- CAPI EMQ 최적화 모름
- "어디든 파는" → 신뢰도 낮음

**우리의 반박:**
> "저희는 MedSpa만 합니다. Botox CPL이 얼마가 정상인지,
> HIFU 시즌이 언제인지, PRP를 Meta에서 광고하면 안 되는 이유가 뭔지
> 저희는 알고 있어요."

---

### Type C: 대형 디지털 에이전시

**특징:**
- 멀티 채널 (Google + Meta + SEO)
- 리포트 화려함
- 낮은 개인화 (AM이 너무 많은 클라이언트 관리)
- 장기 계약 (최소 6-12개월)

**가격:** $3,000-$10,000/월

**약점:**
- 지역 소규모 MedSpa에 맞지 않는 규모
- 유연성 낮음
- AI/자동화 도입 느림
- 리드 후속 자동화 없음

**우리의 반박:**
> "대형 에이전시는 당신이 고객 100명 중 1명이에요.
> 저희는 당신의 비즈니스를 실제로 이해하는 전문 팀입니다."

---

### Type D: 프리랜서 Meta 전문가

**특징:**
- 개인 운영, 저렴한 가격
- 광고 운영만 (자동화 없음)
- 보고 불규칙
- 갑자기 사라질 위험

**가격:** $500-$1,500/월

**약점:**
- GHL 설정 불가
- CAPI 모름
- 예약 추적 없음
- 확장성 없음

**우리의 반박:**
> "프리랜서는 광고는 잘 해도 CRM 자동화와 CAPI 최적화까지는 못 해요.
> 저희는 3개 시스템이 연결된 패키지입니다."

## 4. Meta 광고 라이브러리 경쟁사 조사 방법

**주 1회 실행 SOP:**
```
1. facebook.com/ads/library 접속
2. 국가: United States
3. 광고 유형: 모든 광고
4. 검색: "[도시명] medspa" (예: "Houston medspa")
5. 현재 운행 중인 광고만 필터
6. 분석 항목:
   - Hook 유형 (첫 3초 무엇으로 시작?)
   - 오퍼 (무엇을 주나?)
   - 크리에이티브 (이미지? 비디오? 캐러셀?)
   - 랜딩 페이지 (GHL 사용 여부 확인)
   - 집행 기간 (오래 집행 = 성과 있음 신호)
```

**클라이언트 Audit 시 포함:**
- 경쟁사 3개 광고 스크린샷
- 각 경쟁사 약점 + 우리 클라이언트가 공략할 각도
