# AI 프롬프트 라이브러리 (MedSpa 전용)

> 모든 프롬프트는 Claude API 기준으로 작성. 실제 사용 시 `[BRACKET]` 부분을 클라이언트 정보로 교체.

---

## SECTION 1: 광고 카피라이팅

### PROMPT-AD-01: Meta Lead Form 광고 카피 생성

```
당신은 미국 MedSpa를 전문으로 하는 퍼포먼스 마케터입니다.
아래 정보를 바탕으로 Meta Lead Form 광고 카피 3가지 버전을 만들어주세요.

## 클라이언트 정보
- 클리닉 이름: [CLINIC_NAME]
- 위치: [CITY, STATE]
- 타겟 시술: [TREATMENT]
- 주요 타겟: 30-55세 여성
- 현재 오퍼: [OFFER - e.g., "Free Consultation + $50 Credit"]
- 차별화 포인트: [USP]

## 작성 기준
- 각 버전마다 다른 Hook 사용: (1) Pain Point, (2) Before/After, (3) Social Proof
- 본문 3-4문장, 헤드라인 1문장, CTA 1문장
- 미국 영어, 자연스럽고 따뜻한 톤
- 과장 없이, FDA 규정 준수 (의학적 효과 단정 금지)

## 출력 형식
각 버전:
**HOOK TYPE:** [유형]
**HEADLINE:** [헤드라인]
**BODY:** [본문]
**CTA:** [CTA]
**WHY THIS WORKS:** [한 줄 설명]
```

### PROMPT-AD-02: 시즌별 오퍼 카피 (계절 캘린더)

```
MedSpa 마케터로서 [MONTH] 시즌에 맞는 [TREATMENT] 광고 오퍼를 만들어주세요.

## 시즌 컨텍스트
- 월: [MONTH]
- 계절적 이벤트: [e.g., "Valentine's Day", "Summer Prep", "Holiday Season"]
- 타겟 고민: [e.g., "Valentine's Day 전 피부 결 개선"]

## 오퍼 조건
- 할인율: [X%] 또는 [$X off]
- 기간 제한: [DEADLINE]
- 수량 제한: [e.g., "선착순 15명"]

다음을 만들어주세요:
1. 광고 헤드라인 3개 (A/B/C 테스트용)
2. 리드폼 Thank You 화면 메시지 1개
3. 예약 확인 SMS 템플릿 1개

톤: 설레고 따뜻하게, 절박하지 않게
```

### PROMPT-AD-03: Before/After 비디오 스크립트

```
30초 Instagram Reel / TikTok 스타일 MedSpa 광고 스크립트를 써주세요.

시술: [TREATMENT]
Hook 방향: Before/After 변화 스토리 (실제 고객 관점)

구조:
[0-3초] HOOK: 시각적 임팩트 + 텍스트 오버레이
[3-10초] PROBLEM: 타겟 고객의 공감 포인트
[10-20초] SOLUTION: 시술 설명 (의학적 표현 최소화)
[20-30초] CTA: 오퍼 + 행동 유도

각 섹션에 대해:
- 화면에 보여야 할 것 (Visual)
- 음성/자막 (Copy)
- 배경 음악 제안 (Vibe)

출력 언어: 영어 (미국)
```

## SECTION 2: 주간/월간 리포트 AI 분석

### PROMPT-RPT-01: 주간 퍼포먼스 분석 + 개선 제안

```
당신은 MedSpa 전문 디지털 마케팅 애널리스트입니다.
아래 지난 7일 성과 데이터를 분석하고, 클라이언트에게 보낼 주간 리포트를 작성해주세요.

## 이번 주 데이터
- 총 광고 지출: $[SPEND]
- Meta 리드 수: [LEADS]
- GHL 등록 연락처: [GHL_CONTACTS]
- 예약 완료: [BOOKINGS]
- 방문 완료: [SHOWS]
- CPL: $[CPL]
- CPB: $[CPB]
- Show Rate: [SHOW_RATE]%

## 지난 주 데이터 (비교용)
[LAST_WEEK_METRICS]

## 분석 요청 사항
1. 이번 주 성과 요약 (2-3문장, 클라이언트가 이해하기 쉽게)
2. 주목할 변화 (개선/악화 항목과 원인 추정)
3. 다음 주 권장 액션 3가지 (구체적으로)
4. 클라이언트 동기부여 멘트 1문장

형식: 클라이언트 직접 전달용 (전문 용어 최소화)
언어: 영어
```

## 월간 데이터

- 총 광고 지출: $[SPEND]

- 총 예약: [BOOKINGS] 건
- 총 방문: [SHOWS] 건
- 평균 시술 단가: $[AVG_TICKET]

## 스토리 구성 (4파트)

```

1. 현재 가장 많이 검색되는 시술 Top 5
2. 각 시술의 타겟 고객층과 주요 관심사

4. 계절적 요인 (현재 [MONTH] 기준)
5. 우리 클라이언트가 즉시 활용 가능한 콘텐츠 아이디어 3가지

출력 형식: 마케터가 바로 실행 가능한 액션 아이템 중심
```
