# AI Automation Agency 비즈니스 모델

> **핵심 명제:** 우리는 광고를 대신 운영해주는 게 아니라, MedSpa 클라이언트의 **리드-투-예약 시스템**을 자동화해주는 회사다.

---

## 1. 전통 에이전시 vs AI 자동화 에이전시

| 구분 | 전통 에이전시 | **AI 자동화 에이전시** |
|---|---|---|
| 포지셔닝 | "광고 대행" | **"예약 자동화 시스템"** |
| 핵심 제품 | 캠페인 운영 | 자동화 시스템 + 캠페인 |
| 노동 구조 | AM 1명 당 5-8 클라이언트 | AM 1명 당 20+ 클라이언트 |
| 보고 방식 | 주 1회 수동 보고 | 매일 자동 Telegram 대시보드 |
| 최적화 | 주 1-2회 수동 점검 | CAPI 실시간 + 일일 AI 분석 |
| 가격 정당화 | 시간 투자 | **결과(예약 수)** |
| 진입 장벽 | 낮음 | GHL + Meta CAPI 셋업 전문성 |
| 확장성 | 선형 (사람 비례) | **비선형 (시스템 비례)** |

## 2. 서비스 패키지 설계

### Starter — $1,500/월 + $2,000 셋업
**포함:**
- Meta Ads 운영 (1 캠페인, $1k-$3k 예산)
- GHL 계정 셋업 + Lead Sync 자동화
- 일일 자동 Telegram 리포트
- 월 1회 전략 콜

**적합:** 신규 오픈 MedSpa, 광고 처음 시작

---

### Growth — $2,500/월 + $2,500 셋업
**포함:**
- Meta Ads 운영 (3 캠페인, $3k-$8k 예산)
- GHL 전체 자동화 (Lead → 예약 → 리마인더 → 리뷰)
- Meta CAPI 연동 (Schedule 이벤트)
- 주간 AI 분석 리포트 + 개선 제안
- 월 2회 전략 콜
- AI 광고 카피 월 8개 초안 제공

**적합:** 월 $5k+ 매출 목표 클리닉

---

### Scale — $4,000/월 + $3,000 셋업
**포함:**
- Meta Ads 운영 (무제한 캠페인, $8k-$20k 예산)
- 멀티 로케이션 GHL 관리
- 전체 자동화 스택 (Lead → Booking → Review → Referral)
- Meta CAPI + 오프라인 전환 추적
- 일일/주간/월간 자동 리포트
- 콘텐츠 하베스팅 + AI 크리에이티브 제작
- 무제한 전략 지원

**적합:** 멀티 로케이션, 월 $20k+ 매출 목표

## 3. 핵심 차별화 요소 (Unique Mechanism)

### "예약 보장 시스템" (BAS: Booking Automation System)

우리가 파는 것은 광고가 아니라 **3-Layer Booking Machine**:

```
Layer 1: ATTRACT    Meta Ads (AI 카피 + 타겟팅)
              ↓
Layer 2: CONVERT    GHL 즉시 SMS + 예약 링크 자동화
              ↓
Layer 3: OPTIMIZE   Meta CAPI → 예약하는 유저로 알고리즘 학습
```

**클라이언트에게 설명하는 방식:**
> "다른 에이전시는 리드를 넘겨드립니다. 저희는 예약이 채워지는 시스템을 만들어드립니다. 리드가 들어오면 2분 안에 자동으로 문자가 가고, 예약이 잡히면 Meta가 그 데이터를 학습해서 다음 달 광고가 더 좋아집니다."

## 4. 세일즈 프로세스

```
1. Prospect 발굴     → Instagram DM, Cold Email, 로컬 MedSpa 검색
2. Discovery Call    → "현재 리드 후속 처리를 어떻게 하세요?" (Pain 파악)
3. Audit Offer       → 현재 Meta Ads + GHL 상태 무료 진단
4. 제안서 발송       → ROI 계산 포함 (예: "월 $2,500 투자 → 예약 40건 → 매출 $20k")
5. Onboarding        → GHL 셋업 → Meta Webhook 등록 → CAPI 연결
6. First Win         → 2주 내 첫 예약 자동화 성공 → 케이스 스터디 제작
```
